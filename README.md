 Yazya Scrap Trading Company – Website

This repository contains the source code for the Yazya Scrap Trading Company website.  
It is a clean, responsive business website built using HTML, CSS, and JavaScript .

 Technologies Used
- HTML5 – Structure and content  
- CSS3 – Custom styling and responsive layout  
- JavaScript – Used only where required  
- Bootstrap  – For layout and components 

 Pages Included
The website includes the following main sections:

- Home – Overview of the company  
- About – Company background and introduction  
- Services – List of scrap trading services offered    
- Contact – Contact information and enquiry details
